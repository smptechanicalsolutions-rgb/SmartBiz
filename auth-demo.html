📝 How It Works

// The auth-guard.js automatically protects pages: Protected Pages (auto-redirect if not logged in): • home.html • invoice.html • settings.html • personal-info.html • dashboard.html • And more... Public Pages (accessible to everyone): • login.html • signup.html • forgot.html • Terms, Privacy, Help, etc. // Add to any protected page: <script src="assets/js/auth-guard.js"></script> // Check authentication in your code: if (authGuard.isLoggedIn()) { // User is authenticated const user = authGuard.getCurrentUser(); console.log('Welcome', user.fullName); }
Key Features:
  • ✅ Session-based authentication (stored in localStorage)
  • ✅ Session expiration handling (24 hours or 7 days with "remember me")
  • ✅ Animated modal when trying to access protected pages without login
  • ✅ Smooth page transitions with CSS animations
  • ✅ User-friendly error messages and redirects
  • ✅ Mobile responsive design
  • ✅ Works with existing auth-service.js
