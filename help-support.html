Help & Support

Quick answers for invoices, GST documents, inventory, and the menu catalog builder.

Last updated: 24 May 2026

Getting started

  • Open Home and use Quick Actions to create invoices, quotations, or delivery challans.
  • Complete Personal Information (company name, GSTIN, address, logo) before your first real invoice.
  • Practice with sample documents TEST-INV-001 … TEST-PRO-001 under Home → All Transactions.
  • Read the full User Guide for step-by-step instructions.

Common tasks

  • Tax invoice: Home → New Invoice → fill party & items → Preview → Download PDF.
  • Quotation → Invoice: Home → Convert Document → pick saved quotation → target Invoice.
  • Price list / menu: Sidebar → Menu — demo food catalog loads on first visit; edit categories and items on the left, preview on the right.
  • Currency display: Settings → Currency — amounts are stored in ₹; USD/EUR use live exchange rates.
  • Watermark: Settings → Invoice → enable watermark and set text (e.g. PAID, ORIGINAL).

Data & backup

Most data is stored in your browser (localStorage) and, when logged in, synced via the app server where configured. Use Settings → Download backup to export products, parties, invoices, and settings as a JSON file. Keep backups on your device or cloud drive.

Account & login

Sign in from Home or Settings to sync preferences. Use Logout on shared computers. Forgot password: use the link on the login page.

Report a problem

When contacting support, include: device (phone/PC), browser, what you clicked, and a screenshot if possible. Do not send passwords or full GSTIN of real customers in email unless required.
